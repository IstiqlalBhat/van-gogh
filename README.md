### What's actually under the hood?

I didn't want to overcomplicate things with a million massive frameworks. It's just:

*   **HTML & CSS** (The bones and the makeup)
*   **Three.js** (The magic sauce that makes the images wave and breathe)
*   **A bit of GLSL** (Okay, this is the scary math part, but it's what creates that trippy distortion effect)

It's pretty lightweight. I kept it that way so I could focus on the visuals without fighting a build tool every five minutes. We've all been there, right?

### Want to run it?

It's super simple. Since there's no complex backend, you just need to serve the files.

If you've got Python installed (and who doesn't these days?), just pop open your terminal in this folder and type:

```bash
python -m http.server
```

Then head over to `localhost:8000` in your browser.
